## AIXUE

修改自 **[lolisaikou](https://github.com/lolisaikou) / [LazyStudy](https://github.com/lolisaikou/LazyStudy)** 自动化脚本，解放你的双手！

####  环境依赖

- 安卓版本`7.0`及以上版本
- 授权 `无障碍` 和 `悬浮窗`。
